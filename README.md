# Google Reviews Widget

A lightweight and customizable **Google Reviews Widget** that allows businesses to display real-time Google reviews on their websites. This widget enhances credibility, trust, and engagement by showcasing authentic customer feedback.

## Features

- ✅ Fetches and displays real Google reviews dynamically
- ✅ Responsive design for all screen sizes
- ✅ Customizable layout and styling
- ✅ Easy integration with any website (HTML, PHP, JavaScript)
- ✅ API-based data retrieval for accuracy

## Installation

### 1. Clone the Repository
```sh
git clone https://github.com/IAmAbdulHafiz/google-reviews-widget.git
cd google-reviews-widget
```
### 2. Get a Google Places API Key
- Go to the Google Cloud Console
- Create a new project or select an existing one
- Enable the Places API
- Generate an API Key

### 3. Configure the API Key
- Open the configuration file (e.g., config.php, .env  and set environment variables)
- Replace YOUR_GOOGLE_API_KEY and YOUR_GOOGLE_PLACE_ID with your actual API Key and Place ID in .env file

## Usage
### 1. Add the Widget to Your Website
- Embed the widget by including the script in your HTML file. For example, if you are using PHP, you can include your widget file:
```sh
<iframe src="https://yourdomain.com/path/to/reviews-widget.php" style="border: none; width: 100%; height: 800px;"></iframe>
```

### 2. Customize Styling
- Modify the CSS within the widget file or your external stylesheet (styles.css) to match your website’s design.
